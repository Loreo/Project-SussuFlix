How to run ?
====================

You will need Npm and Node on your computer :

* [Npm](https://www.npmjs.com/)
* [Node](https://nodejs.org/)


Open a terminal
---------------

    git clone https://github.com/Loreo/Project-SussuFlix.git
    cd Project-SussuFlix
    npm install
    npm start


On your web browser
---------
*   [localhost:8080](http://localhost:8080)
