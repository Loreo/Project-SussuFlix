How to run ?
====================

You will need Npm and Node on your computer :

* [Npm](https://www.npmjs.com/)
* [Node](https://nodejs.org/)


Open a terminal
---------------

    git clone https://github.com/Loreo/React-Movies.git
    npm install
    npm start


On your web browser
---------
*   [localhost:3000](http://localhost:3000)
